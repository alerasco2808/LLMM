5 Conceptos y Propiedades CSS Esenciales

1. La propiedad `display: flex` (Flexbox)

Caso de Uso:

Se utiliza para diseñar estructuras unidimensionales (en una fila o una columna). Es perfecto para alinear elementos, distribuirlos en el espacio disponible y centrarlos verticalmente con facilidad.

Ejemplo de Código:


.contenedor-flexible {
    display: flex;          /* Activa Flexbox */
    justify-content: center; /* Centra los elementos horizontalmente */
    align-items: center;     /* Centra los elementos verticalmente */
    height: 150px;
    background-color: #eee;
}

2. La propiedad `display: grid` (CSS Grid)

Caso de Uso:

Es el sistema de diseño más potente para la web, diseñado para estructuras bidimensionales (filas Y columnas al mismo tiempo). Permite crear layouts complejos estilo "periódico" o galerías de imágenes fácilmente.

Ejemplo de Código:


.contenedor-grid {
    display: grid;
    /* Crea 3 columnas de igual ancho */
    grid-template-columns: repeat(3, 1fr); 
    /* Espacio de 10px entre celdas */
    gap: 10px; 
}

3. La propiedad `transform` (Rotación fija)

Caso de Uso:

Permite modificar el espacio de coordenadas del modelo visual. Se usa para rotar, escalar, inclinar o mover elementos. Es útil para crear elementos decorativos.

Ejemplo de Código:


.elemento-diamante {
    width: 80px;
    height: 80px;
    background-color: #e74c3c;
    /* Rota el elemento 45 grados permanentemente */
    transform: rotate(45deg);
    margin: 30px auto; /* Centrado */
}

4. La propiedad `box-shadow`

Caso de Uso:

Añade efectos de sombra alrededor del marco de un elemento. Se utiliza mucho en diseños modernos para dar sensación de profundidad, elevación o para destacar tarjetas y botones.

Ejemplo de Código:


.tarjeta-con-sombra {
    /* offset-x | offset-y | blur-radius | color */
    box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2);
    border-radius: 8px;
}

5. Funciones de Gradiente (Linear Gradient)

Caso de Uso:

Permite crear una transición progresiva entre dos o más colores. Es muy utilizado en fondos modernos o botones llamativos sin necesidad de usar imágenes externas.

Ejemplo de Código:


.fondo-gradiente {
    height: 100px;
    /* Crea un degradado de azul (#3498db) a morado (#8e44ad) */
    background: linear-gradient(90deg, #3498db, #8e44ad);
    color: white;
    font-weight: bold;
    display: flex;
    justify-content: center;
    align-items: center;
}
